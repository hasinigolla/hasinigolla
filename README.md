<sub>⚙️ This animates automatically once you add the <a href="https://github.com/Platane/snk">Platane/snk</a> GitHub Action to this repo (2-minute setup, workflow below) — turns your real contribution graph into a snake eating your commits.</sub>

<details>
<summary><b>⚡ snake.yml workflow (click to expand)</b></summary>

```yaml
name: generate snake
on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch: {}
  push:
    branches: [ main ]

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: hasinigolla
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
